Each of the following links demonstrates a different database query:

  • Query 1: Displays platform name, platform model, and manufacturer for each platform/model (demonstrates two-table join)
  • Query 2: Displays all games, release dates, game developers, and game publishers (demonstrates three-table join)
  • Query 3: Displays all publishers and their parent companies, only for publishers that have parent companies (demonstrates self-join)
  • Query 4: Counts number of publishers in database (demonstrates aggregate function)
  • Query 5: Displays each user ID alongside the count of how many games they own, for owners of more than 3 games (demonstrates aggregate function using GROUP BY and HAVING)
  • Query 6: Displays all models of all PlayStations (demonstrates text-based search query using LIKE with wildcard(s))
  • Query 7: Displays game title and genre for all games with more than 2 genres (demonstrates subquery)
  • Query 8: Displays all games alongside how many copies the game has sold (based on tracked users) (demonstrates stored function)
  • Query 9: Takes an input of manufacturer (i.e. Sony, Microsoft), and returns how many consoles they've manufactured (demonstrates stored procedure)
  • Query 10: If you try to add a platform and the release date is after 2022, it automatically sets that to be 2021 (demonstrates trigger)
